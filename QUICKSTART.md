# RadLog 快速開始

## 5 分鐘上手

### 1. 下載並安裝
- Windows: 下載 `RadLog.exe`，雙擊執行
- macOS: 下載 `RadLog.app`，拖到 Applications

### 2. 首次登入
1. 點「用 Google 登入」
2. 選擇你購買時用的 Google 帳號
3. 授權 RadLog 存取 Google Sheets

### 3. 建立你的 Sheet
1. 開啟 [Google Sheets](https://sheets.google.com)
2. 建立新試算表，命名為「RadLog」
3. 第一行輸入標題：`時間 | 病歷號 | 分類 | 備註`
4. 複製 Sheet URL 中的 ID（在 `/d/` 和 `/edit` 之間）

### 4. 設定 RadLog
1. 點 RadLog 的「設定」
2. 貼上你的 Spreadsheet ID
3. 儲存

### 5. 開始使用！

**快捷鍵：** `Ctrl+Shift+R`（Windows）或 `Cmd+Shift+R`（macOS）

**快速輸入格式：**
```
病歷號, 分類, 備註
```

**範例：**
```
1234567, tumor, liver mass suspect HCC
A9876543, vascular, AAA 5.2cm stable
B1111111, infection, lung abscess improving
```

---

## 常用分類建議

| 分類 | 說明 |
|------|------|
| tumor | 腫瘤相關 |
| vascular | 血管相關（AAA, dissection 等）|
| infection | 感染相關 |
| trauma | 外傷相關 |
| followup | 需要追蹤 |
| interesting | 教學案例 |
| urgent | 緊急通知臨床 |

你可以自訂分類，輸入新的分類時會自動新增。

## 快捷操作

| 動作 | 快捷鍵 |
|------|--------|
| 開啟 RadLog | `Ctrl+Shift+R` |
| 送出記錄 | `Enter` |
| 開啟 Sheet | 點選單「開啟 Sheet」|

---

## 疑難排解

### 「找不到 Spreadsheet」
- 確認 Spreadsheet ID 正確
- 確認 Sheet 已分享給你的 Google 帳號

### 「授權失敗」
- 確認使用購買時的 Google 帳號
- 嘗試重新登入

### 快捷鍵沒反應
- Windows: 確認沒有其他程式占用 Ctrl+Shift+R
- macOS: 確認已授權「輔助使用」權限
